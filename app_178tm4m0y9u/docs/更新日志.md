## 2026-07-05 v2.0.6 血统技能专属化 + 宠物派遣奇遇系统

### 一、血统技能专属化（按 `docs/技能.log`）

- **新增 `PET_BLOODLINE_DEX` 对象**（`config.js`）：为约 225 个宠物定义独立的血统技能 `name` 与 `desc`
  - 涵盖所有 T 级（T1 弱阶 / T2 普通 / T3 中阶 / T4 强阶 / T5 顶阶）+ 融合限定 + 初级野生宠物
  - 每个宠物都有独特的血统技能名称与描述，例如 `小火焰: 炎鳞迸发`、`冰晶兽: 寒晶反哺`
- **`generatePetBloodlineSkill` 修改**：优先使用 `PET_BLOODLINE_DEX[pet.name]` 的专属 name/desc
  - 战斗效果仍按专长预设 `BLOODLINE_EFFECT_PRESETS` 计算（保持战斗可计算性）
  - 专属描述会拼接专长标签 + 当前效果数值 + 进阶强化提示
  - 融合限定宠物与血统珠植入宠物保持原血统，不受专属化影响
- 修改血统技能名称/描述只需修改 `PET_BLOODLINE_DEX`，无需改动其他代码

### 二、宠物派遣奇遇系统

#### 核心机制
- **闲置宠物派遣**：派遣至已通关的地图区域探索，最多 3 只组队
- **战力门槛**：派遣宠物总战斗力需达到地图 `minPower` 标准才能开始
- **战力加成**：超出最低战力每 10% 额外 +5% 奖励倍率（封顶 ×3.0）
- **探索时长**：可选 1/4/8 小时（倍率 1.0/4.5/10.0）
- **基础产出**：金币、经验、养成材料；低概率（5%×战力倍率）掉落稀有道具

#### 配置（`config.js`）
- `DISPATCH_DURATIONS`：3 种时长与倍率
- `DISPATCH_MAPS`：11 个地图派遣区域，按 `MAPS` 顺序配置
  - mapId 1 起源草地（minPower 500）→ mapId 11 终焉神殿（minPower 250,000）
- `DISPATCH_BASE_REWARD`：基础金币 1000 / 经验 500 / 材料 2 个 / 稀有概率 5%
- `DISPATCH_POWER_BONUS_PER_10PCT = 0.05`
- `DISPATCH_MAX_SLOTS = 3`：同时派遣上限

#### 状态字段（`state.js`）
- `G.dispatches`：当前进行中的派遣列表
- `G.dispatchHistory`：已领取的派遣历史（保留最近 50 条）
- `G.dispatchDailyUsed`：每日派遣计数（仅记录，不硬性限制）
- 旧存档自动迁移

#### 核心逻辑（`economy.js`）
- `getDispatchMapConfig(mapId)`：获取地图配置
- `isMapUnlockedForDispatch(mapId)`：地图是否已通关（`mapProgress[mapId].phase === 'cleared'`）
- `getAvailableDispatchPets()`：可派遣宠物（不在出战阵容、不在其它派遣中）
- `getDispatchTotalPower(petIds)`：总战斗力（`getPetCombatPower` 求和）
- `getDispatchPowerBonus(total, min)`：战力超额加成倍率
- `getDispatchProgress(dispatch)`：进度 0~1（基于真实时间）
- `getDispatchExpectedReward(dispatch)`：预计奖励预览
- `startDispatch(mapId, petIds, durationIdx)`：开始派遣（含完整校验）
- `collectDispatch(dispatchId)`：领取奖励（完成时）
- `recallDispatch(dispatchId)`：强制召回（无奖励）
- `applyDispatchRareDrop(rareId)`：稀有掉落应用（rare_egg/blood_orb_*/skill_random）

#### UI（`ui.js`）
- 活动页新增 `dispatch` tab：`🎒 派遣奇遇`
- `renderActivityDispatch()` 渲染函数：
  - 进行中的派遣列表（进度条、预计完成时间、领取/召回按钮）
  - 新建派遣面板（地图选择、宠物多选、时长选择、战力检查、奖励预览）
  - 派遣历史（近 10 条）
  - 派遣收获日志
- 交互函数：`_dispatchPickMap`、`_dispatchPickDuration`、`_dispatchTogglePet`、`_dispatchStart`、`_collectDispatch`、`_recallDispatch`

#### 日常任务（`config.js`）
- 新增 `dispatch_finish` 任务：完成 1 次派遣奇遇，奖励 20 钻石

### 三、平衡说明
- 派遣系统天然受限于：3 并发槽位 + 1/4/8 小时时长，不设硬性每日次数限制
- 稀有掉落概率随战力加成提升，但封顶 50%
- 派遣宠物可获得 30% 经验奖励，鼓励派遣主力外的宠物
